# DevLab

Development workspace for exploration, experiments, and in-progress notebooks.

Nothing here is production: code that matures graduates into `util/` (libraries)
or `GetFMPData/` (entry points). The folder previously named
`MarketInternalMonitor` was retired on 2026-06-13; its legacy notebooks,
`featureStore.py`, and old universe CSVs live in the git-ignored
`legacyFiles/` folder at the repo root (reference only — stale paths, do not run).
